How are results brought?
  • Find Features
  • Specific Event

Every event you'll see listed on the results section meet at least all the following characteristics:

  • Public
  • Live status
  • Event Start Date within a timeframe of 1 to 6 months from now.
  • Sales End Date at least 1 month from now.
  • Events are sorted by the amount of tickets sold (descending)

These constraints aim to provide more compelling examples by bringing popular events that will still be live for a while.

Any additional filters (Country, Category, Features, etc.) you might add will be applied on top of these constraints.

PRO TIP: Make sure you double check features are visible (not hidden) before sharing them by taking a look on the event listing page and/or the Admin link.

You'll be able to support attendees by searching for upcoming events even with only a few clues.

Event Name, Organizer Name, Attendee Name and Venue Location fields support partial inputs.

E.g:

By entering "Cooking" on the Event Name field, you'll be able to find the "2020 World Championship Barbecue Cooking Contest" event.

Or, by entering "Riverside" on the Venue Location field, you'll be able to find an event with the following location: "Tom Lee Park 357 Riverside Dr Memphis, TN 38103"
